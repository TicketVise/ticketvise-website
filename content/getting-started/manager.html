Creating an inbox

Creating an inbox to match your course is as simple as creating a link on the canvas page. When you visit this link on the canvas page, you create the course in TicketVise. When students and agents click this link, they too will be added to the course, with the permissions inherited from canvas.

Configuring an inbox

When opening the inbox, you can go to the configuration to change some settings about the course. Most of these settings are self-explaining, we will quickly go over some of the others. Toggling the "Assignee visible to students" changes if the students can see if and who is assigned to their ticket. Replying on a ticket will however always show the author of the reply. You can set a timer for closing a ticket if no reply has been sent for some amount of time, or you can send an alert if no answer has been given after some time. These help to prevent tickets from being unanswered and keep them organised.

Scheduling

To automatically assign tickets to agents, there are two scheduling methods. For now, workflows are being developed to increase the uses of automatically assigning tickets. Currently, there is an option to manually assign tickets, no automatic scheduling takes place. The least assigned first makes sure that every agent is assigned the same amount of tickets, if agent 1 that get a ticket from agent 2 assigned to them will not get another one assigned to them before agent 2 has received a new ticket to compensate for the loss of the first. When using round robin, this will not happen, but all tickets will be assigned to the next ta in the list.

Configuring users

The users tab on the configuration page shows all users connected to the course known by TicketVise. As a manager, you can change the permissions of each user in the course and view information about them.

Configuring labels

Labels can be added to the wish of the manager. Labels are useful for organizing the tickets inside the course. Changing the visibility of the student or enabling the label in general are ways in wich you can control the use of the labels. Labels not visible for students can be used to organise labels within the staff.

Inbox statistics

The statistics from the inbox gives insight in the amount of tickets and the activity of the agents. These insights can help to recognise where the problems lies with the students as well as the agents.
