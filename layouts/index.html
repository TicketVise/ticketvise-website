Frequently asked questions

I don't like our current mailinglist.

We totally get it and that's exactly the reason we build TicketVise. We streamline the process of students asking questions.

How does it work?

Students ask questions which land on a clean board to track the questions per course. This gives you a clear overview of which questions need to be answered.

Can I integrate TicketVise with my LMS?

Yes! We currently only support Canvas but are expending into other platforms. If you have a specific platform you would like, let us know!

How does TicketVise contribute to didactic skills of students?

We offer statistics for the teachers which they can use to get insight into questions. For example you can see which type of questions are asked the most.

I don't want to assign each question myself.

We neither! TicketVise therefore automates this process with two different scheduling algorithms. The tickets can be assigned automatically in a Round Robin fashion or can be assigned to the teaching assistant with the least amount of tickets.

Do I need another platform to discuss questions with my team?

No! The goal of Ticketvise is to optimize answering questions and discussing them is a important aspect of it. TicketVise therefore supports discussing a ticket internally and allows the staff members to mention each other.
