Frequently asked questions

I don't like our current mailinglist.

We totally get it and that's exactly the reason we build TicketVise. We streamline the process of students asking questions.

How does it work?

Students ask questions which land in a clean over board to track the questions per course. This gives you a clear overview of which questions need to be answered.

Can I integrate TicketVise with my LMS?

Yes! We currently only support Canvas but are expending into other platforms. If you have a specific platform you would like, let us know!

How does TicketVise contribute to didactic skills of students?

We offer statistics for the teachers which they can use to get insight into questions. For example you can see which questions are asked the most.

I don't want to assign each question myself.

We neither! TicketVise therefore includes streamlining the process using workflows. Which automates the process of assigning questions to assistants.

Can I discuss the questions with the rest of the team?

Yes, we can chat directly in TicketVise with the team. You can even mention people or another question.
